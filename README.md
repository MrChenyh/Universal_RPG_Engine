# 万界剧情模拟引擎 (Universal RPG Engine) 🌌

基于 OpenClaw 架构开发的文字 RPG 引擎。只需投喂任意小说 (.txt/.md) 或原创大纲，即可全自动生成带有难度分级、外挂系统、以及飞书云端属性面板的沉浸式文字游戏。

## ✨ 核心特性
- 📦 **全自动量子切片**：支持百万字长篇小说导入，自动建立剧情索引，拒绝 Token 爆炸。
- 🎭 **命运模式分级**：支持【爽文模式】（自带外挂不死）、【普通模式】与【苦难模式】。
- 📊 **飞书云端托管**：玩家面板、背包物品、NPC好感度全自动同步至飞书文档，聊天界面极简清爽。
- 🎲 **极高自由度**：自定义开局身世，每一回合支持完全自定义行动。

## 🚀 安装方法
1. 确保已安装并配置好 OpenClaw 与 ClawX。
2. 使用 Skills CLI 安装：
   `openclaw skills install ~/.openclaw/skills/universal-rpg-engine`
   *(注：或者在 ClawX 界面中点击“安装技能”导入此文件夹)*
3. 重启网关：`openclaw gateway restart`
4. 在对话框中输入 `/skills` 确认加载，随后发送“开始游戏”或直接上传小说 txt 文件即可。
